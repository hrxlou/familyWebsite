# 👨‍👩‍👧‍👦 우리 가족 (FamilyWebsite Premium)

가족들만의 소중한 사진, 게시글, 그리고 일정을 공유하고 소통할 수 있는 프리미엄 패밀리 웹 앱입니다. 토스(Toss)의 정갈하고 프리미엄한 디자인 시스템을 모티프로 하여, 현대적이고 직관적인 사용자 경험을 제공합니다.

---

## ✨ 핵심 기능

### 1. 📝 가족 게시판
*   **소중한 기록**: 텍스트와 이미지를 활용해 가족의 일상을 남겨보세요.
*   **커뮤니케이션**: 댓글을 통해 서로의 글에 반응하고 따뜻한 대화를 나눌 수 있습니다.
*   **스마트 검색**: 키워드 기반 검색으로 과거의 기록을 빠르게 찾아보세요.

### 2. 📸 프리미엄 가족 앨범
*   **슬라이드 뷰**: 라이트박스 기능을 통해 고화질로 사진을 감상하세요.
*   **추억 저장소**: 가족과 함께한 순간들을 카테고리별로 깔끔하게 관리합니다.

### 3. 📅 가족 캘린더 & 일정
*   **스마트 캘린더**: 주요 행사, 생일, 기념일을 한눈에 확인하세요.
*   **알림 위젯**: 메인 화면에서 다가오는 일정(D-Day)을 실시간으로 확인하여 중요한 날을 놓치지 않게 도와줍니다.

### 4. 🤖 스마트 가이드 챗봇
*   **이용 안내**: 사이트 사용법이나 궁금한 점을 묻고 답변을 받을 수 있는 지능형 챗봇이 탑재되어 있습니다.

### 5. 🌙 프리미엄 UI/UX
*   **다크 모드**: 눈의 피로를 덜어주는 고급스러운 다크 모드를 완벽 지원합니다.
*   **글래스모피즘**: 세련된 유리 질감 효과와 부드러운 애니메이션으로 프리미엄한 느낌을 줍니다.
*   **PWA 지원**: 스마트폰 홈 화면에 추가하여 앱처럼 편리하게 사용할 수 있습니다.

---

## 🛠 기술 스택

*   **Backend**: Python / Flask
*   **Database**: SQLite / SQLAlchemy
*   **Frontend**: Vanilla HTML5, CSS3 (Modern Utility-first CSS), JavaScript (ES6+)
*   **Features**: PWA (Service Worker), Chatbot API, File Upload System

---

## 🚀 빠른 시작

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```
서버 실행 후 브라우저에서 `http://127.0.0.1:5000`으로 접속하세요.

---

## 📂 프로젝트 구조
*   `app.py`: 통합 백엔드 코어 및 라우팅
*   `models.py`: 데이터베이스 스키마 정의
*   `static/`: 디자인 시스템(CSS), 클라이언트 로직(JS) 및 리소스
*   `routes/`: 기능별(Auth, Board, Album, Calendar) 모듈화된 라우트
