# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.0   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it privately. Do not open a public issue.

To report a vulnerability, please contact the maintainer through the GitHub profile or email directly if listed. We aim to respond within 48 hours and provide a fix as soon as possible.
